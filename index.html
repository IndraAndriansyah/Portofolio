Case Study: Investigation and Detection of Network Traffic

The report presents a comprehensive analysis of network traffic using the Wireshark application and the cybersecurity analysis website VirusTotal.com, encompassing two distinct case studies. In Case 1, anomalies in TCP traffic were identified, indicating a potential SYN flood attack. In Case 2, suspicious activity was detected through an excessive number of DNS queries and the use of Domain Generation Algorithm (DGA), which may suggest the occurrence of a DNS Tunneling attack.
